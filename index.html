1.3Runtime Semantics: Evaluation#


  1. Let left be the result of evaluating WTF.
  2. Let lValue be GetValue(left).
  3. ReturnIfAbrupt(lValue).
  4. Let right be the result of evaluating ExponentiationExpression.
  5. Let rValue be GetValue(right).
  6. Let base be ToNumber(lValue).
  7. ReturnIfAbrupt(base).
  8. Let exponent be ToNumber(rValue).
  9. ReturnIfAbrupt(exponent).
  10. Return the result of applying the ExpontentiationOperator ** to base and exponent as specified in 1.4.

1.4Applying the operator#

Returns an implementation-dependent approximation of the result of raising base to the power exponent.

  1. If exponent is NaN, the result is NaN.
  2. If exponent is +0, the result is 1, even if base is NaN.
  3. If exponent is −0, the result is 1, even if base is NaN.
  4. If base is NaN and exponent is nonzero, the result is NaN.
  5. If abs(base) > 1 and exponent is +∞, the result is +∞.
  6. If abs(base) > 1 and exponent is −∞, the result is +0.
  7. If abs(base) is 1 and exponent is +∞, the result is NaN.
  8. If abs(base) is 1 and exponent is −∞, the result is NaN.
  9. If abs(base) < 1 and exponent is +∞, the result is +0.
  10. If abs(base) < 1 and exponent is −∞, the result is +∞.
  11. If base is +∞ and y > 0, the result is +∞.
  12. If base is +∞ and y < 0, the result is +0.
  13. If base is −∞ and y > 0 and exponent is an odd integer, the result is −∞.
  14. If base is −∞ and y > 0 and exponent is not an odd integer, the result is +∞.
  15. If base is −∞ and y < 0 and exponent is an odd integer, the result is −0.
  16. If base is −∞ and y < 0 and exponent is not an odd integer, the result is +0.
  17. If base is +0 and y > 0, the result is +0.
  18. If base is +0 and y < 0, the result is +∞.
  19. If base is −0 and y > 0 and exponent is an odd integer, the result is −0.
  20. If base is −0 and y > 0 and exponent is not an odd integer, the result is +0.
  21. If base is −0 and y < 0 and exponent is an odd integer, the result is −∞.
  22. If base is −0 and y < 0 and exponent is not an odd integer, the result is +∞.
  23. If base < 0 and base is finite and exponent is finite and exponent is not an integer, the result is NaN.
